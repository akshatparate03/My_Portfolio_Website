Portfolio Website

This is a fully responsive personal portfolio website built using HTML, CSS, and JavaScript. It features a dark/light theme toggle, allowing users to switch based on preference. The site includes my resume (view and download option), a showcase of my projects with live demo and source code links, a section for certificates, a display of skills, and links to my LeetCode and GitHub profiles. A contact form is integrated, which sends messages directly to my Gmail inbox, enabling seamless communication.
