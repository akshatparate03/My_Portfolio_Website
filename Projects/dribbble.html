Dribbble Clone

Creating a Dribbble website clone using HTML and CSS involves replicating the visual structure and styling of the original Dribbble platform, focusing entirely on the frontend. This includes designing a responsive navigation bar with logo and menu links, a stylish hero section with a search bar and category filters, and a grid layout to showcase design shots with hover effects and interaction icons like likes and views. Using modern CSS techniques such as Flexbox, Grid, and media queries ensures the layout adapts to different screen sizes, closely mimicking the clean, professional look of Dribbble. This project helps improve UI design skills and deepens understanding of responsive web design using just HTML and CSS.
